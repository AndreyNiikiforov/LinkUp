1. Кто мы

LinkUp — это личный проект, созданный для общения между пользователями. Мы уважаем вашу приватность и защищаем ваши персональные данные.

2. Какие данные мы собираем

  • Номер телефона — необходим для регистрации и идентификации пользователя
  • Имя пользователя (username) — для отображения в чатах
  • Сообщения — текстовые, голосовые, видеосообщения и файлы, которые вы отправляете
  • Информация об устройстве — браузер, операционная система, IP-адрес

3. Как мы используем данные

  • Для обеспечения работы мессенджера (доставка сообщений, звонки)
  • Для идентификации пользователей при входе
  • Для улучшения качества сервиса
  • Для технической поддержки

4. Передача данных третьим лицам

Мы НЕ передаём ваши персональные данные третьим лицам, за исключением случаев, предусмотренных законодательством Российской Федерации.

5. Хранение данных

  • Все данные хранятся на защищённых серверах
  • Мы используем шифрование для защиты передаваемых данных
  • Вы можете удалить свой аккаунт в любой момент

6. Ваши права

  • Получить информацию о своих данных
  • Исправить неточные данные
  • Удалить свой аккаунт

7. Контакты

По вопросам, связанным с вашими персональными данными, вы можете связаться с нами:

  • Email: linkup@support.ru
