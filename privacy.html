Политика конфиденциальности

Последнее обновление: 14 марта 2026

1. Кто мы

LinkUp — это личный проект, созданный для общения между пользователями. Мы уважаем вашу приватность и защищаем ваши персональные данные.

2. Какие данные мы собираем

  • Номер телефона — необходим для регистрации и идентификации пользователя
  • Имя пользователя (username) — для отображения в чатах
  • Сообщения — текстовые, голосовые и видеосообщения, которые вы отправляете
  • Информация об устройстве — браузер, операционная система, IP-адрес
  • Данные о сессиях — информация о ваших входах в аккаунт

3. Как мы используем данные

  • Для обеспечения работы мессенджера (доставка сообщений, звонки)
  • Для идентификации пользователей при входе
  • Для улучшения качества сервиса
  • Для технической поддержки
  • Для обеспечения безопасности и предотвращения мошенничества

4. Передача данных третьим лицам

Мы НЕ передаём ваши персональные данные третьим лицам, за исключением случаев, предусмотренных законодательством Российской Федерации.

5. Хранение данных

  • Все данные хранятся на защищённых серверах
  • Мы используем шифрование для защиты передаваемых данных
  • Вы можете удалить свой аккаунт в любой момент через настройки профиля
  • Сообщения хранятся для обеспечения непрерывности общения

6. Ваши права

  • Получить информацию о своих данных (напишите нам на почту)
  • Исправить неточные данные (в настройках профиля)
  • Удалить свой аккаунт (в настройках устройств)
  • Отозвать согласие на обработку данных (удалив аккаунт)

7. Защита данных

Мы принимаем все необходимые меры для защиты ваших данных от несанкционированного доступа, изменения, раскрытия или уничтожения. Однако помните, что ни один метод передачи данных через Интернет не является 100% безопасным.

8. Изменения в политике

Мы можем обновлять эту политику конфиденциальности. Актуальная версия всегда доступна по этой ссылке. Продолжая использовать LinkUp после изменений, вы соглашаетесь с обновлённой политикой.

9. Контакты

По вопросам, связанным с вашими персональными данными, вы можете связаться с нами:

  • Email: linkup.privacy@support.ru
